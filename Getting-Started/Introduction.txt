What bootstrap is and why should learn it ?
--------------------------------------------------------
-open source front end framework for fast web development
-pre made html and css classes 
-javascript widgets and pluggins
-mobile first and responsive layout

Advantages for bootstrap
-----------------------------------
-browser compatiblity
-Faster application development
-Good for developers who lack designing skills or UI skills
-Easy to learn  
-Responsive layout
